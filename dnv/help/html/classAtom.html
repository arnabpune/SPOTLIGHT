Detailed Description

This is the Atom class. It is the most fundamental building-block for molecule generation using DeNovo. An atom object contains all the force-field parameters (see: ForceField) and position data contained in it. The Molecule class uses arrays of these Atom* (pointers) and defines connections between them in order to manage molecules All position information is stored in double precision and the unit of distance in DeNovo is nm

This class makes use of the Eigen libraries. See Eigen: http://eigen.tuxfamily.org/ for more details. (git repo: https://gitlab.com/libeigen/eigen.git)
