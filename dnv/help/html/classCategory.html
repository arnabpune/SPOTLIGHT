Detailed Description

This class is used to store lists of atom-type names grouped under a category name. A single atom-type can be grouped under multiple categories.
The format and working are briefly explained the in algorithm PDF.
An example category line looks like this:

halogen FA|F1|F2|F3|CLAL|CLAR|BRAR|BRAL|I

This is the "halogen" Category.
