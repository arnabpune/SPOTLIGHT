Detailed Description

This is the (abstract) Container class
It allows the user to design many shapes of containers as long as distances to various points can be computed as required by the virtual functions (see functions of this class)
The initial intent of this class is to provide a container to restrain molecule growth (as an alternative to using restraints to hotspot residues)
See Also: Cuboid, Sphere
