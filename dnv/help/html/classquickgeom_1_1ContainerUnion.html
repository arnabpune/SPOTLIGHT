Detailed Description

The ContainerUnion class allows to take the 'union' of multiple containers as a region. Any point contained in ANY ONE of the containers will be in the union.
The containing box (See Container::getContainingBox()) can sphere (See Container::getContainingSphere()) will contain the entire region.
