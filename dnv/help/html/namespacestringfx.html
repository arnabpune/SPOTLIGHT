Find the index of any character from a list (given as a string) in another string. Priority is given to the earlier characters in the string.

Character priorities are decided by order in the list string. The earlier characters have higher priority.
The index of a character lower in priority will be returned only if no characters of higher priority exist in the string.

split a string by a given delimiter and generate a std::vector of strings as output

This function also allows "protector characters" where a string between a start protector and end protector remains uncut
For example ((1,2,3),(4,5,6)) when split over ',' will yield (1,2,3),(4,5,6) if '(' and ')' are used as protector charactes in the start and end positions respectively.
