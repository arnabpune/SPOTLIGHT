Detailed Description

This is a simple struct to store the components of an angle (three atoms). The actual order in which the atoms are provided is of no significance as long as the angle subtended is accurately described (eg. The middle atom must be the second atom)
It also calculates the plane of the three atoms and the angle between them when they are provided
