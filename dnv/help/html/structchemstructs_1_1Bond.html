Detailed Description

This is a simple struct to store the components of a bond (three atoms). It also calculates the vector of the bond when the component atoms are provided
This class makes use of the Eigen libraries. See Eigen: http://eigen.tuxfamily.org/ for more details. (git repo: https://gitlab.com/libeigen/eigen.git)
