\doxysubsection{Detailed Description}
This is the \mbox{\hyperlink{classAtom}{Atom}} class. It is the most fundamental building-\/block for molecule generation using De\+Novo. An atom object contains all the force-\/field parameters (see\+: \mbox{\hyperlink{classForceField}{Force\+Field}}) and position data contained in it. The \mbox{\hyperlink{classMolecule}{Molecule}} class uses arrays of these Atom$\ast$ (pointers) and defines connections between them in order to manage molecules All position information is stored in double precision and the unit of distance in De\+Novo is nm

This class makes use of the Eigen libraries. See Eigen\+: \href{http://eigen.tuxfamily.org/}{\texttt{ http\+://eigen.\+tuxfamily.\+org/}} for more details. (git repo\+: \href{https://gitlab.com/libeigen/eigen.git}{\texttt{ https\+://gitlab.\+com/libeigen/eigen.\+git}})
