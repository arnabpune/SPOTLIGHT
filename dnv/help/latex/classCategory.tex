\doxysubsection{Detailed Description}
This class is used to store lists of atom-\/type names grouped under a category name. A single atom-\/type can be grouped under multiple categories.~\newline
The format and working are briefly explained the in algorithm PDF. ~\newline
An example category line looks like this\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{halogen FA|F1|F2|F3|CLAL|CLAR|BRAR|BRAL|I}

\end{DoxyCode}
