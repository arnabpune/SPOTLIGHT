\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classSystem}{System}} class is designed envisaging a true physical \char`\"{}system\char`\"{} with multiple interacting components.~\newline
 It is this object that also stores ambient condition data (such as temperature), and the multiple components in this \mbox{\hyperlink{classSystem}{System}} are expected to \char`\"{}interact\char`\"{} with each other~\newline
 From a more algorithmic perspective, all \mbox{\hyperlink{classMolecule}{Molecule}} objects in a given \mbox{\hyperlink{classSystem}{System}} have non-\/bonding interaction energies calculated between them (during generation). The exception to this is that two macromolecules (\mbox{\hyperlink{classProtein}{Protein}} class objects), if present in the same system do not have interaction energies calculated automatically (you can specifically perform this time-\/taking calculation)~\newline
 This algorithm assumes all atoms of a macromolecule to be stationary during generation.~\newline
 See algorithm PDF for more details on generation.
