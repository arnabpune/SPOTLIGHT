\doxysubsection{Detailed Description}
This is the (abstract) \mbox{\hyperlink{classquickgeom_1_1Container}{Container}} class~\newline
 It allows the user to design many shapes of containers as long as distances to various points can be computed as required by the virtual functions (see functions of this class)~\newline
 The initial intent of this class is to provide a container to restrain molecule growth (as an alternative to using restraints to hotspot residues)~\newline
 See Also\+: \mbox{\hyperlink{classquickgeom_1_1Cuboid}{Cuboid}}, \mbox{\hyperlink{classquickgeom_1_1Sphere}{Sphere}}
