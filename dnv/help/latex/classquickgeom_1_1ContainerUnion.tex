\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classquickgeom_1_1ContainerUnion}{Container\+Union}} class allows to take the \textquotesingle{}union\textquotesingle{} of multiple containers as a region. Any point contained in ANY ONE of the containers will be in the union.~\newline
 The containing box (See \mbox{\hyperlink{classquickgeom_1_1Container_a6b75d074354c0cf66a94c9b62ec7b1f2}{Container\+::get\+Containing\+Box()}}) can sphere (See \mbox{\hyperlink{classquickgeom_1_1Container_aa673b931c7dbe2023c69d0684dc39c91}{Container\+::get\+Containing\+Sphere()}}) will contain the entire region.~\newline
