\doxysubsection{Detailed Description}
The forcefx namespace contains all the force-\/calculation functions. The force is simply a derivative of the potential (See documentation PDF for a reference to the potentials used). 

This namespace makes use of the Eigen libraries. See Eigen\+: \href{http://eigen.tuxfamily.org/}{\texttt{ http\+://eigen.\+tuxfamily.\+org/}} for more details. (git repo\+: \href{https://gitlab.com/libeigen/eigen.git}{\texttt{ https\+://gitlab.\+com/libeigen/eigen.\+git}})
