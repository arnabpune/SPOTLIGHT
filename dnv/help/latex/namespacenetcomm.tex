\doxysubsection{Detailed Description}
\begin{DoxyRefDesc}{Deprecated}
\item[\mbox{\hyperlink{deprecated__deprecated000026}{Deprecated}}]netcomm nammespace \end{DoxyRefDesc}


This namespace deals with the data being passed between threads when using external charges.~\newline
 In this case, the generating thread (the thread that completes a generation successfully) adds it\textquotesingle{}s newly generated molecule (along with the \mbox{\hyperlink{classSystem}{System}} object in which it was generated) to a queue. ~\newline
 The queue is processed molecule by molecule by another asynchronous thread, which clears the queue parallely as it is filled.
